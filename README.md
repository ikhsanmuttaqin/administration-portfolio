# Administration Portfolio

This repository showcases practical administrative systems built using Microsoft Excel and Word. Each project simulates real-world business scenarios and demonstrates skills in finance, human resources, and task management.

## Projects

### 1. Cashflow Management System
A system to track and monitor daily financial transactions and monthly summaries.

### 2. Employee Management System
A structured employee database with attendance and leave tracking.

### 3. Task Tracking System
A task and project tracking tool for managing work progress and deadlines.

## Tools Used
- Microsoft Excel
- Microsoft Word

## Purpose
This portfolio demonstrates my ability to design and manage administrative systems in a professional work environment.
